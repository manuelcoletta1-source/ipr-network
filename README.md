# IPR Network

Public visibility surface for the **IPR Network**: an EU-first operational identity infrastructure based on append-only evolution and deterministic verification.

## What this repository publishes

- Network homepage: `/`
- Network status page: `/status/`
- Node registry: `/nodes/registry.json`

## Properties

- Append-only continuity
- Hash-only public exposure (no data custody)
- Deterministic verification
- Fail-closed model
- EU-first / Audit-first / GDPR-min
